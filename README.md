## Project Description
RadarScope is a mobile weather application that provides users with real-time weather radar and forecast information in a simple and easy-to-use interface. The app solves the problem of complicated weather applications by delivering only the most necessary weather data in a clean and accessible format. It is designed for everyday users who want quick and accurate weather updates directly from their mobile device.
